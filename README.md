# Agenda Megamaxi

Esta pasta organiza a agenda por projetos, tarefas pendentes e tarefas concluídas.

## Estrutura

- `AI/`: agentes e documentações relacionadas à IA.
- `Pendentes/`: tarefas abertas, sempre em arquivos Markdown (`.md`).
- `Projetos/`: um arquivo `<nome-do-projeto>.md` para a descrição de cada projeto.
- `Concluídos/`: tarefas finalizadas, com a mesma convenção de nomes de `Pendentes/`.

## Convenção de nomes das tarefas

As tarefas em `Pendentes/` e `Concluídos/` devem seguir um destes formatos:

```text
pai_nome_da_tarefa_dd-mm-aaaa.md
nome_da_tarefa_dd-mm-aaaa.md
nome_da_tarefa_SD.md
```

- `pai`: nome do projeto associado. É opcional.
- `nome_da_tarefa`: descrição curta usando `_` no lugar de espaços.
- `dd-mm-aaaa`: data no formato brasileiro.
- `SD`: sem data definida.

Exemplos:

```text
megamaxi_revisar_catalogo_15-07-2026.md
organizar_documentos_SD.md
```
